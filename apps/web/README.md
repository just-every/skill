# Every Skill Frontend

`apps/web` is a frontend-only catalog for Every Skill.

It contains only public pages:
- `/` homepage
- `/skills` catalog, benchmark visibility, and local recommendation demo

The UI is static-data-first and does not require runtime Worker API calls.

## Commands

```bash
pnpm --filter @justevery/web test
pnpm --filter @justevery/web build
```

## Data source

Catalog/benchmark/recommendation data is loaded from:

- `apps/web/src/data/catalog.ts`

## Rendering

Prerender output is generated by:

- `apps/web/scripts/prerender.tsx`

Routes prerendered:

- `/`
- `/skills`
